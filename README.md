CodingChick's Udemy
===================
Coding Chick's Udemy is an unofficial Windows Phone 8.1 client for Udemy developed solely by CodingChick (Efrat Barak). 

To install CodingChick's Udemy please visit the [Windows Phone Store](http://www.windowsphone.com/en-us/store/app/codingchick-udemy/f4ea6edc-b3a6-4569-9f21-864965a04470?signin=true)

Udemy has thousands of courses and more than 3,000,000 people who’ve chosen to partake in their training. Using this app you can participate and watch online courses, training, and tutorials. 

The app features:
- Finding interesting courses based on availability, sales and categories.
- Checking out course information such as rating, user reviews, description and much more. 
- Watching videos for courses you own and sample lectures from most other courses.
- Courses on a variety of topics such as: Programming (java, python, php, ruby, and more), Entrepreneurship, Photography, Yoga, Marketing, Salsa, Cake Decorating, Design and more!

NOTE: This app requires you have an existing Udemy user and doesn’t support purchasing new courses in-app.
